Library Management System in Java (Maven)

Overview
This Java-based Library Management System is a console application designed to manage library operations such as adding and 
removing books, registering and deleting members, borrowing and returning books, and tracking fines for late returns. It provides 
separate interfaces for librarians and members to interact with the library system.

->Features

#Librarian Interface

-Register new library members.
-Remove existing library members.
-Add books to the library inventory.
-Remove books from the library inventory.
-View all library members, including their fine amounts.
-View all books in the library inventory.


#Member Interface

-List all available books in the library.
-List books currently borrowed by the member.
-Borrow books from the library.
-Return borrowed books to the library.
-Pay fines for overdue books.
